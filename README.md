# The Munching Queen Website

Website Astro statis bilingual untuk The Munching Queen.

```bash
npm install
npm run dev
npm run build
npm run preview
```

Build production tersedia di `dist/` dan bisa dipasang pada Vercel, Netlify, atau Cloudflare Pages.

Konten produk: `src/data/products.ts`. Design system: `DESIGN_SYSTEM.md`. Alamat sengaja belum ditampilkan.
